 <h2>Getting Started</h2>

<b>Installation</b>

1.  Clone the repo onto your local machine

```sh
 git clone https://github.com/finHop/FinanceTracker.git
```

2.  Install dependencies

```sh
npm install
```

3.  Run the app

```sh
npm run dev
```
